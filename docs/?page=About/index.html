About

We want to provide a central location for all console-related documentation with a focus on emulating the systems. Basically, this means linking to other sites providing the information, but if they disappear from the web (as we have seen happening quite often) we provide a mirror for the files.

Of course, we are not aiming at getting other people's credit. We link to the original sites, and we mention the original author of a document (if we know about them). If you know of updates or additional documents which we should include, or have a correction for an existing link, please mail us at editors@emu-docs.org.
